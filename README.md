# TripDriver Backend API

Backend API for a **Self-Drive Car Rental System**.

## Tech Stack

* .NET 8
* ASP.NET Core Web API
* Entity Framework Core
* SQL Server
* JWT Authentication
* Swagger (OpenAPI)

## Roles

* **Customer** – Browse cars, create bookings, pay deposit
* **Owner** – Manage cars and confirm rentals
* **Admin** – Manage system and bookings

## Main Features

* User authentication with JWT
* Car listing and management
* Booking and rental workflow
* Deposit payment handling
* Booking status tracking
